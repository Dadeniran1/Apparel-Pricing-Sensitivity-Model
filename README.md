# Apparel-Pricing-Sensitivity-Model

## Objective: Pricing and cost sensitivity model to analyze how changes in price, demand, and input costs impact contribution margin and operating profit

## Tools Used: Microsoft Excel, Power BI, Cost Analysis

## Questions Answered: How does inflation affect cost and margins. What price maximizes profit? How do margins change under different scenarios?

## Key Insights: Lower inflation cost results in larger operational margins. Pricing the hoodies at $100 would produce the smallest revenue, however, the largest profit.
